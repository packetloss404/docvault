// Features

Everything you need to manage
documents at scale

21 tightly integrated apps covering the full document lifecycle — from ingestion to retention.

📄

14-Plugin Ingestion Pipeline

Ordered plugin chain from preflight to post-consume. Each plugin runs setup, process, and cleanup with stop-on-failure support.

👁

OCR with Multiple Modes

Tesseract + OCRmyPDF with skip/redo/force modes. Automatic deskewing, rotation correction, and image cleaning. Outputs PDF/A for long-term archival.

📷

Thumbnail Generation

Automatic WebP thumbnail generation at configurable resolution (default 400x560). Supports PDF, image, and Office documents via PyMuPDF.

☷

Barcode Detection & ASN

Scans up to 5 pages at 300 DPI with 2x upscaling for QR codes, Code128, and patch-T separators. Auto-assigns Archive Serial Numbers.

🌐

Language Detection

Automatic language identification using langdetect. Sets the document language field for locale-aware processing and search.

📅

Date Extraction

Intelligent date parsing from document content using dateparser. Supports multiple formats and localizations for automatic document dating.

📂

Multi-Format Parsing

Parse PDF, images, Office docs, email (.msg), and HTML. Content extraction with BeautifulSoup, pikepdf, and extract-msg.

🔧

Pre/Post-Consume Hooks

Execute custom shell scripts before and after document processing. Configurable timeout with environment variable injection.

🤖

ML Document Classification

Four scikit-learn MLP classifiers for auto-assigning tags, correspondents, document types, and storage paths. Hash-based change detection for incremental training.

💬

AI-Powered Chat (RAG)

Ask questions about individual documents or across your entire library. Retrieval-Augmented Generation with chunk-level cosine similarity ranking.

💡

AI Summarization & Titles

Auto-generate document summaries in 2-4 sentences. AI-suggested titles based on content analysis. Supports OpenAI and self-hosted Ollama.

🔭

FAISS Vector Store

Semantic document search with FAISS IndexFlatIP. L2-normalized embeddings with cosine similarity. Persistent index with disk serialization.

🔎

Hybrid Search (RRF)

Combines Elasticsearch keyword search with FAISS semantic search using Reciprocal Rank Fusion. Configurable keyword/semantic weight balance.

🎓

Named Entity Recognition

spaCy-powered NER extraction: persons, organizations, locations, dates, monetary amounts. Entities indexed in Elasticsearch for faceted search.

▭

Zone OCR Templates

Define bounding box regions on document pages for targeted field extraction. Supports text, date, number, and currency types with regex validation.

📈

Elasticsearch Full-Text Search

Multi-match with fuzziness across title, content, and filename. Field-specific queries (tag:, type:, correspondent:), faceting, and date histograms.

⚙

State Machine Engine

Full workflow engine with states, transitions, entry/exit actions, and conditional evaluation. AST-validated safe expression engine for conditions.

⚡

7 Action Backends

Document property updates, tag management, cabinet assignment, metadata modification, email sending, webhook calls, and workflow launching.

⏰

Escalation Rules

Time-based auto-transitions when workflows stall. Priority-ordered with configurable deadlines (minutes, hours, days, weeks) and condition checks.

📖

Retention Policies

Per-document-type auto-trash and auto-delete with configurable time periods. Supports days, months, and years for compliance requirements.

🔔

Multi-Channel Notifications

In-app, email, and webhook notification channels. Per-user event preferences. Real-time delivery via Django Channels WebSocket.

📐

Workflow Rules & Triggers

Document-triggered and consumption-triggered workflow automation. Rules with condition evaluation for auto-launching workflows on matching documents.

📊

Usage Quotas

Per-user, per-group, or global quotas on document count and storage bytes. Priority inheritance: user > group > global.

🔍

Search Analytics & Curations

Track query history, click-through rates, response times. Admin-curated pins, hides, and field boosts. Synonym management for query expansion.

✍️

E-Signature Workflows

Sequential and parallel signing orders. UUID-tokenized public signing pages. Email/access-code verification. PDF certificate of completion generation.

📝

8 Annotation Types

Highlight, underline, strikethrough, sticky note, freehand drawing, rectangle, text box, and rubber stamp (APPROVED, REJECTED, CONFIDENTIAL, etc.).

💬

Comments & Discussion

Threaded markdown comments on documents with annotation replies. Soft-deletion for moderation. Real-time updates via WebSocket.

🔗

Share Links

Public document sharing with optional password protection (SHA-256), expiration dates, and download tracking. Original or archive (PDF/A) file versions.

🔒

Check-out / Check-in

Exclusive document locking with configurable expiration and auto-release. Upload blocking during checkout to prevent conflicts.

🚀

Contributor Portal

Branded public upload portals with slug-based URLs. Document requests with tokenized links. Rate limiting, file type restrictions, and review workflows.

📋

Document Relationships

Typed relationships: supersedes, references, attaches-to, duplicates, and custom types. Directional graph with interactive visualization.

📃

Version Control

Full version history with active version tracking. Multi-file support per document (original, archive, transformed). Diff and rollback support.

⚖

Legal Hold

Preservation orders with DRAFT/ACTIVE/RELEASED lifecycle. Multi-criteria document capture (date range, type, tags, correspondent). Custodian notification and acknowledgement.

📚

Comprehensive Audit Trail

13 auditable action types: login, create, update, delete, download, sign, verify, OTP setup, password change, and more. IP address and user-agent tracking.

🔐

AES-256 Encryption at Rest

PBKDF2 key derivation with 100K iterations. Random IV per file. Transparent encrypt-on-save, decrypt-on-read. Wraps any storage backend.

🔑

OIDC / SSO

OpenID Connect integration with auto-provisioning. Configurable authorization, token, user, and JWKS endpoints. Works with Keycloak, Auth0, Azure AD.

📱

TOTP Two-Factor Auth

Time-based one-time passwords with QR code setup. 8 hashed backup codes for account recovery. Per-device confirmation flow.

🖌

GPG Document Signing

ASCII-armored GPG/PGP signatures with key ID tracking. Cryptographic verification with timestamp. Non-repudiation for compliance.

📦

Physical Records

MPTT location tree: Building > Room > Cabinet > Shelf > Box. Charge-out tracking with expected return dates. Destruction certificates with method and witness documentation.

💻

Object-Level Permissions

django-guardian for per-document ACLs. Role-based access control. IP whitelist/blacklist. Rate throttling (20 anon / 120 user per minute).

💾

Pluggable Storage Backends

Abstract base class with local filesystem, Amazon S3/MinIO, and encrypted storage implementations. All backends support save, open, delete, exists, list, and size.

🧾

Content-Addressable Storage

SHA-256 content hashing with sharded paths (ab/cd/hash). Automatic deduplication with reference counting. Integrity verification and dedup statistics.

☁

S3-Compatible Storage

Full Amazon S3 and MinIO support with configurable endpoint, region, and bucket. boto3-powered with streaming upload/download.

🔒

Encrypted Storage

AES-256-CBC encryption layer wrapping any backend. PKCS7 padding, random IV per file, PBKDF2 key derivation. 64KB chunked streaming for memory efficiency.

📝

Non-Destructive Mode

Original files are never modified. Archive (PDF/A) copies maintain a separate checksum. Parallel storage of originals and processed versions.

📂

Custom Storage Paths

Pattern-based file organization with Jinja2 templates. Auto-assignment via ML classifier for intelligent document filing.

// Deployment

Deploy anywhere in minutes

Docker Compose for development, multi-stage production Dockerfile, and Helm chart for Kubernetes. Your infrastructure, your rules.

🐳 Docker Compose

Full dev stack in one command: PostgreSQL, Redis, MinIO, web server, and Celery worker.

$ git clone https://github.com/docvault/docvault.git $ cd docvault $ docker compose up -d

☸ Kubernetes / Helm

Production-ready Helm chart with web, worker, beat deployments, ingress, configmaps, secrets, and PVCs.

$ helm repo add docvault https://charts.docvault.io $ helm install docvault docvault/docvault \ --set postgresql.enabled=true \ --set elasticsearch.enabled=true

📦 Production Image

Multi-stage Dockerfile: Node.js frontend build, Python dependency layer, and slim production runtime with Gunicorn.

$ docker build -f Dockerfile.production -t docvault:1.0 . $ docker run -p 8000:8000 \ -e DATABASE_URL=postgres://... \ -e REDIS_URL=redis://... \ docvault:1.0

🛠 Environment Config

All settings via environment variables. No config files to manage. django-environ powered.

DJANGO_SECRET_KEY=your-secret-key DATABASE_URL=postgres://user:pass@db:5432/docvault REDIS_URL=redis://redis:6379 STORAGE_BACKEND=s3 LLM_PROVIDER=openai LLM_API_KEY=sk-... ELASTICSEARCH_ENABLED=true
