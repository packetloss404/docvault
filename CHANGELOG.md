# Changelog

All notable changes to DocVault will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-04-15

### Added
- Full document management system with 22 sprint features
- Angular 21 frontend with 68+ components
- Django 5.2 REST API with 21 app modules
- Document ingestion pipeline (PDF, Office, images, email)
- OCR via OCRmyPDF with zone OCR templates
- ML classification (tags, correspondent, document type)
- LLM integration (OpenAI, Ollama, Azure) with semantic search
- Workflow engine with state machines and trigger-action rules
- E-signatures (GPG + request-based) and annotations
- Legal hold with mutation blocking
- Content-addressed storage with deduplication
- Contributor portal for external document submission
- RBAC with admin UI, OIDC/SSO support
- Comprehensive test suite (1,029 backend + 525 frontend tests)
